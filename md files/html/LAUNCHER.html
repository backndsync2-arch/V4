üöÄ Quick Start Instructions

Step 1: Start Backend Server

  1. Open PowerShell or Command Prompt
  2. Navigate to: C:\Users\dolab\Downloads\sync2gear_backend
  3. Run: .\START_SERVER.ps1 (or START_SERVER.bat)
  4. Wait for: Starting development server at http://127.0.0.1:8000/

Step 2: Start Frontend Server (If Node.js is installed)

  1. Open a new PowerShell or Command Prompt window
  2. Navigate to: C:\Users\dolab\Downloads\V2 Sync2gear
  3. Run: npm install (first time only)
  4. Run: npm run dev
  5. Wait for: Local: http://localhost:5173/

Step 3: Test the Application

  1. Click on "Frontend Application" link above (or go to http://localhost:5173)
  2. Click "Sign In" button
  3. Use any of the test credentials above
  4. Explore the dashboard, music library, announcements, scheduler, and zones

Step 4: Test API Directly

  1. Click on "API Documentation (Swagger)" link above
  2. Try the authentication endpoints
  3. Use the "Authorize" button to login with test credentials
  4. Test other API endpoints
